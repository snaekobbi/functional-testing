Requirements

Translation/mapping: support for languages used in the countries of the interested parties

  • The system shall support the braille code used in Denmark, Finland, Norway, Sweden and Switzerland for uncontracted braille.

Support for layout/formatting of paragraphs/text blocks

  • The system shall have support for top, right, bottom and left margins on blocks.

  • The system shall have support for top, right, bottom and left borders and padding on blocks.
